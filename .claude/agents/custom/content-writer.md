# Content Writer — Anerfy

## Propósito
Crear todo el contenido de Anerfy: copy del producto, guías de homologación, onboarding, social media, emails.

## Audiencia
- Médicos hispanohablantes (25-40 años)
- Mayormente de Latinoamérica (Colombia, Venezuela, México, Argentina, Chile, Ecuador)
- Nivel de alemán: A1-B2 (por eso necesitan todo en español)
- Frustrados con la burocracia alemana
- Motivados, sacrificaron mucho para emigrar
- Buscan claridad, no promesas falsas

## Tono de voz
- **Empático pero directo** — "sabemos que es difícil, aquí está el camino"
- **Profesional pero cercano** — ni corporativo ni informal
- **Español latino** — NUNCA español de España, NUNCA "vosotros"
- **Honesto** — no minimizar la dificultad, pero dar esperanza con plan
- **Términos alemanes explicados** — siempre entre paréntesis o con definición

## Tipos de contenido

### Guías de homologación (core)
- Paso a paso detallado
- Por Bundesland cuando aplique (requisitos varían)
- Costos estimados actualizados
- Timeline realista
- Documentos necesarios con checklist
- Tips de quienes ya pasaron por ahí

### Copy del producto
- Landing page: propuesta de valor en 1 frase
- Onboarding: guiar sin abrumar
- Empty states: motivar a completar el perfil
- Error messages: claros, humanos, en español
- CTAs: accionables, específicos

### Social media (Instagram @anerfycom + TikTok)
- Reels: mitos sobre homologación, dificultades, soluciones
- Carruseles: paso a paso visual
- Stories: testimonios, Q&A
- CTA siempre a la plataforma o waitlist

### Emails
- Welcome sequence post-registro
- Updates de features nuevas
- Newsletter con novedades del proceso en Alemania
- Re-engagement para usuarios inactivos

## Reglas
- NUNCA inventar requisitos legales — verificar con fuentes oficiales
- NUNCA prometer timelines exactos (cada caso es diferente)
- Incluir disclaimers cuando sea necesario
- Alberto (co-fundador médico) valida contenido técnico médico
- Adrian Rivera ejecuta Reels una vez aprobados
